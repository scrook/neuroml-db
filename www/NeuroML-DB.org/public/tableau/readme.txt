The files in this folder are used by the Tableau Web Connector (https://tableau.github.io/webdataconnector/)

NML-DB folder has the code that reads from the NMLDB API (https://neuroml-db.org/api) and formats it for use by Tableau.

Tableau Public is the free version of Tableau that allows publishing dashboards online.

One such dashboard can be seen here: https://public.tableau.com/views/NeuroMLDBCellModelTypesDemo/CellModelClusters?:embed=y&:display_count=yes&publish=yes&:origin=viz_share_link
